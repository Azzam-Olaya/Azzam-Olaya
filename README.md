### 📁 Projets Phares

#### 🚀 **MorLancer** – *Plateforme de Freelance 100% Marocaine*
* **Stack :** Laravel, API REST, Vue.js, Docker, PostgreSQL
* **Description :** Conception et développement complet d'une plateforme web permettant la publication, la recherche et la gestion de missions en ligne.
* **Architecture :** 
  * Backend structuré sous forme d'une API REST sécurisée avec gestion avancée des rôles, authentification, messagerie interne et tableaux de bord interactifs.
  * Frontend moderne développé avec Vue.js.
  * Environnement de base de données PostgreSQL conteneurisé et isolé grâce à Docker.

#### 💼 **WorkSphere** – *Virtual Workspace*
* **Stack :** HTML5, CSS3, JavaScript (Vanilla)
* **Description :** Application de gestion dynamique de personnel au sein de différents espaces et salles de travail. 
* *Fonctionnalités clés :* Formulaire d'ajout d'employés avec validation de données stricte (regex pour emails, numéros) et système d'assignation en temps réel selon les rôles.

#### 🎮 **PlaySphere** – *Explore the Best Video Games*
* **Stack :** JavaScript, Fetch API, HTML/CSS
* **Description :** Plateforme d'exploration de jeux vidéo consommant une API externe de manière asynchrone pour afficher des données riches de façon dynamique et responsive.
